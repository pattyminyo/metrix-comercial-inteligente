# 📊 Metrix Comercial Inteligente

Assistente virtual com Inteligência Artificial voltado para gestão comercial, indicadores estratégicos e performance de vendas.

---

## 🚀 Objetivo

Criar uma experiência digital inteligente capaz de responder perguntas, explicar KPIs, realizar simulações comerciais e apoiar decisões estratégicas.

---

## 💡 Funcionalidades

✅ Explicação de indicadores comerciais  
✅ Simulações de metas e vendas  
✅ Insights de performance comercial  
✅ Respostas contextualizadas  
✅ Interface simples e intuitiva  

---

## 📈 KPIs abordados

- Conversão
- Churn
- CAC
- Ticket Médio
- Receita
- Produtividade Comercial
- Meta x Realizado

---

## 🛠 Tecnologias Utilizadas

- Python
- Google Colab
- GitHub
- Inteligência Artificial
- Streamlit (expansão futura)

---

## 🎯 Público-alvo

Gestores comerciais, analistas, supervisores de vendas e profissionais que desejam tomar decisões orientadas por dados.

---

## 🔗 Projeto desenvolvido para desafio DIO.me

Aplicando IA, Python, Dados e UX em uma solução prática de mercado.
